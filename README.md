js-file2string
===============

Installation
------------

```sh
go get github.com/pocke/js-file2string
```

Usage
------

In command line

```sh
$ js-file2string file1 file2 > files.js
```

In JavaScript

```javascript
var files = require('files');
var file1Value = files['file1'];
```
